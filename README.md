# WiFi-Radar
WiFi Radar is a powerful Python-based network scanning and monitoring tool designed to detect nearby WiFi networks, analyze signal strength, and display real-time wireless data in a user-friendly format.  This project is built for educational and ethical cybersecurity purposes only.
👨‍💻 Developer

Coded by Mr. Sabaz Ali Khan

🚀 Features
📶 Scan nearby WiFi networks in real-time
📡 Display SSID, BSSID, Channel, and Signal Strength
⚡ Fast and lightweight performance
🖥️ Clean CLI (Command Line Interface) output
🔍 Detect hidden networks (if supported)
📊 Signal strength visualization (optional upgrade)
🛡️ Designed for ethical hacking learning
🛠️ Technologies Used
Python 3
scapy (packet manipulation)
pywifi (WiFi interface control)
subprocess (system commands)
time and os modules
